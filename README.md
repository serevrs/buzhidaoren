# 老牛软路由管理工具

一个用于管理 MikroTik 路由器的图形界面工具。

## 功能特点

- 路由器连接管理
- 系统状态监控
- 路由表管理
- 脚本管理和执行

## 安装要求

- Python 3.11 或更高版本
- 依赖包：
  - PyQt5
  - routeros-api
  - humanize

## 使用方法

### 方式一：直接运行（推荐）
1. 下载 `老牛软路由管理工具.exe`
2. 双击运行即可
3. 在连接界面输入路由器信息：
   - IP地址（默认：192.168.88.254）
   - 用户名（默认：houtai）
   - 密码（默认：houtai）

### 方式二：从源码运行
1. 确保已安装 Python 3.11
2. 安装依赖包：
   ```bash
   pip install -r requirements.txt
   ```
3. 双击运行 `run_gui.bat`

## 注意事项

- 请确保路由器已开启 API 功能
- 建议使用管理员权限的账户登录
- 首次运行可能需要允许防火墙访问
- 如果 Windows 提示安全警告，点击"仍要运行"即可
